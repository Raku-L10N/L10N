NAME
====

L10N::HU - Hungarian localization of Raku

SYNOPSIS
========

```raku
use L10N::HU;
mond "Hello World";
```

DESCRIPTION
===========

L10N::HU contains the logic to provide a Hungarian localization of the Raku Programming Language.
